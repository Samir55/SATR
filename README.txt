please run the following command in the website dir to view the browser

# For Python version returned is 3.X
------------------------------------------

# On Linux/MacOS
	python -m http.server 8090

# On Windows, try 
	"python -m http.server 8090" or "py -3 -m http.server 8090"

# If Python version returned is 2.X
------------------------------------------
	python -m SimpleHTTPServer 8090


The webpage can be opened on any browser using the following URL:

http://localhost:8090/
